#Keepfast CLI

Keepfast in your command line with reporting.

 [![npm version](https://badge.fury.io/js/keepfast-cli.svg)](https://badge.fury.io/js/keepfast-cli)

## Install
```
$ npm i keepfast-cli -g
```

## Run
```shell
$ keepfast https://www.fellipe.com/
```
